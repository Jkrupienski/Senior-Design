* File > Settings > Lang & Framework > Flask > press to see check box > 'Apply'
press F12 > Console in website for debugging

1. create virtual environment once:
   > python -m venv venv

2. activate virtual environment:
   > .\venv\Scripts\activate

3. install dependencies:
   > pip install flask flask-sqlalchemy flask-login flask-bcrypt numpy pandas opencv-python

4. run flask server (MOST RECENT (old is: python app.py):
   > flask run
